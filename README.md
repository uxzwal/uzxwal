# Ujjwal Kumar — DevOps Portfolio

> "6 months in. Shipping like a senior. The graph doesn't lie."

A premium, mobile-optimized portfolio for **Ujjwal Kumar**, Junior DevOps Engineer.  
Design philosophy: **Neural Control Dashboard** meets Stripe + Linear quality.

## 🚀 Features

- **3D Cloud Node Cluster** — Interactive canvas with 12 floating nodes + bezier connections
- **Neural Dashboard** — GitHub contribution graph + live system status cards
- **Zero Trust Pipeline** — Interactive CODE → SAST → SBOM → SIGN → OPA → DEPLOY flow
- **Tech Stack Array** — Icon grid with hover glow effects
- **Project Cards** — 3D tilt glassmorphism cards for Overdrive + High Availability
- **Certifications** — Floating achievement badges
- **Dark/Light Mode** — Seamless theme toggle
- **Custom Cursor** — Glowing neon cursor (desktop only)
- **Particle Background** — Connected particle network canvas

## 📦 Quick Start

This is a **zero-dependency** static site. Just open it:

```bash
# Option 1: Open directly
open index.html

# Option 2: Serve locally
npx serve .
# or
python3 -m http.server 8080
```

## 🌐 Deploy to Vercel / Netlify

```bash
# Vercel
npx vercel --prod

# Netlify
npx netlify deploy --dir . --prod
```

Or drag-and-drop the folder to [netlify.com/drop](https://app.netlify.com/drop).
